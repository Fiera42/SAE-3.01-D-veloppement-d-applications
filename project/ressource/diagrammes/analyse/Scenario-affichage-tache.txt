Affichage :

L'utilasteur lance l'application 
Le systeme lui propose diférents affichage
L'utilisateur choisit ensuite l'affichage bureau 
Le systeme affiche la page bureau ou l'on peut voir les diférentes tâche ansi que les collones
L'utilisateur clique sur une des taches.
Le systeme va afficher cette tâche en mode dévlopper.
L'utilisateur va cliquer sur le bouton retour. 
Le système retourne à l'affichage bureau. 
L'utilisateur fait glisser une tache dans une autre colonne.
Le systeme va afficher le changement en direct dans le bureau.
L'utilisateur ferme l'application.





Ajout : 

L'utilasteur lance l'application 
Le systeme lui propose diférents affichage
L'utilisateur choisit ensuite l'affichage bureau 
Le systeme affiche la page bureau.
L'utilisateur clique sur ajouter une tache 
Le systeme montre la page de creation de tache.
L'utilisateur rentre les donnés nécésaire à la création de la tache (titre , description , collaborateur)
Le system va creer la tache correspondante et l'ajouter au bureau.
L'utilisateur ferme l'application.
